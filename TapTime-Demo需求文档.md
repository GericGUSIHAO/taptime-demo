# TapTime Demo 需求文档

> **产品名称**：TapTime（时刻）
> **产品形态**：移动端 App（Demo 用交互式网页展示）
> **一句话定位**：时间的账本。只记大账，不记零碎。

---

## 一、核心功能

### 1.1 页面一：今日

**功能**：记录 + 展示今天的时间花费

```
┌─────────────────────────────────┐
│                                 │
│  7月8日  周二                    │
│                                 │
│  今天                           │
│                                 │
│  ┌───────────────────────────┐  │
│  │  💼 工作    2h 30min       │  │
│  │  📖 阅读    45min          │  │
│  │  🎮 游戏    2h             │  │
│  │  ☕ 休息    30min          │  │
│  └───────────────────────────┘  │
│                                 │
│  共记录  5h 45min               │
│                                 │
│  ─────────────────────────      │
│                                 │
│         [ 轻点记录 ]             │
│                                 │
└─────────────────────────────────┘
```

**记录交互**：

```
点"轻点记录" → 底部滑出：

┌─────────────────────────────────┐
│                                 │
│  在做什么？                       │
│                                 │
│  💼 工作   📖 阅读   🏃 运动      │
│  ☕ 休息   🍽 吃饭   🎮 游戏      │
│                                 │
│  多久？                          │
│                                 │
│  15m  30m  45m  1h  1.5h  2h   │
│                                 │
│         [ 记上 ]                │
│                                 │
└─────────────────────────────────┘
```

## 二、设计规范

### 2.1 视觉风格

**Editorial Magazine 极简风**

| 色彩 | 色值 | 用途 |
|---|---|---|
| 米白底 | #FAF9F7 | 背景 |
| 纯黑 | #1A1A1A | 主文字 |
| 深赭石 | #C45C2E | 强调色 |
| 浅灰 | #E5E5E5 | 分割线 |
| 中灰 | #999999 | 次要文字 |

### 2.2 字体

| 字体 | 用途 |
|---|---|
| Fraunces | 大标题、数字 |
| Space Grotesk | 正文、UI 文字 |

### 2.3 动效

| 动效 | 触发 | 时长 |
|---|---|---|
| 数字递增 | 记录时间块时 | 0.3s |
| 卡片点击反馈 | 点击分类卡片 | 0.2s |
| 页面切换 | Tab 切换 | 0.3s |
| 滚动渐入 | 元素进入视口 | 0.4s |

## 五、验收标准

| 项 | 标准 |
|---|---|
| 记录功能 | 选分类 + 选时长 + 确认，三步完成 |
| 今日展示 | 清晰列出今天的时间花费 |
| 本周展示 | 条形图 + 每日标记 |
| 数据持久化 | 刷新页面数据不丢失 |
| 动效 | 核心交互有平滑动效 |
| 设计 | 米白底 + 赭石强调色，极简高级 |
