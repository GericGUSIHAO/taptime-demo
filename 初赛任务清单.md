## 二、可用设计/前端 Skill 清单

| 阶段 | 使用哪个 Skill | 做什么 |

| **设计探索** | huashu-design | 出3版不同设计风格的高保真原型，选最好的 |

## 五、Demo 开发任务（网页版）

### 产品定位
- 产品形态：App（但 Demo 用网页展示）
- Demo 形式：交互式网页，模拟手机界面
- 核心功能：时间块记录 + 习惯打卡（反向打卡）
- 设计风格：Editorial Magazine 极简风 + 仪式感

### 功能范围（MVP）

| 功能 | 说明 | 优先级 |
|---|---|---|
| 时间块记录 | 点一下卡片 = 15分钟，点两下 = 30分钟 | ✅ 核心 |
| 时间块展示 | 显示今天/本周记录的时间块 | ✅ 核心 |
| 习惯打卡 | 建立期打卡，稳定期反向打卡 | ✅ 核心 |
| 习惯状态切换 | 30天后自动切换为"默认已完成" | 🔮 可选（初赛可简化） |
| 数据统计 | 简单的时间分布可视化 | 🔮 可选 |

### 技术实现
- 用 TRAE IDE + 设计 Skill 开发
- 纯前端网页（HTML/CSS/JS）
- 可以用本地存储模拟数据持久化
- 设计成移动端适配（模拟手机 App 界面）
